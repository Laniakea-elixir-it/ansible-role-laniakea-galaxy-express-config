What is Laniakea?

Laniakea is a Galaxy on-demand platform that lets you provision customizable Galaxy environments on heterogeneous cloud infrastructures, simplifying deployment, configuration, and management.

Laniakea@ReCaS is the ELIXIR-IT managed service hosted at the ReCaS-Bari data center: it provides cloud resources to create project-specific Galaxy instances with dedicated tools, workflows, and reference data.
